# WhatToEat

## 功能点分析

- 用户需要登录（邮箱登录，后面创建账号）- 用邮箱发验证码
- Form 表单收集用户喜好，（风格/面食/提供早中饭三餐的选择）
- 重置用户所有的设定，
- 主页面，一个按钮
- 反馈（可选）
